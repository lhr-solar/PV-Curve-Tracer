# PV-Curve-Tracer
Source code for running the PV Curve Tracer Board.

Used pins:
| Schematic Pin No.  | uC Pin Designation | Function    |
| ------------------ | ------------------ | ----------- |
| L1                 | PA9                | LED1        |
| L2                 | PA10               | LED2        |
| L4                 | GND                | GND         |
| L5                 | PA12/CAN1_TD       | CAN TX      |
| L7                 | PB7                | LED3        |
| L13                | PA11/CAN1_RD       | CAN RX      |
| R2                 | GND                | GND         |
| R4                 | 5V                 | 5V          |
| R5                 | PA2                | ADC V       |
| R6                 | PA7                | ADC C       |
| R8                 | PA5                | DAC Control |
| R14                | 3.3V               | 3.3V        |
